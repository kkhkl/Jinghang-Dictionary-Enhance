景行词库【增强版】

## 📖 项目说明

该项目是为 **Google Gboard** 制作的自定义短语词库，旨在提供更上手的打字体验并提升输入效率。此版本适用于需要更全面的输入、超出日常使用范围的词汇量，相较于标准版拥有更细致、小众的词汇类别，在语言用词、中文文化、文学诗句方面有所更进一步扩展。

- 词库文件为 **Gboard 导出的 `.zip` 格式**  
- 可在 Gboard 自带的导入入口直接导入使用  
- 该词库为自带词库的增强，不会与底层词库冲突  

---

## 🛠 支持方案

| 输入方案 | 描述 |
----------------
| 中文全拼 | 完整中文拼音输入 |

| 小鹤双拼 | 双拼方案，适合熟悉小鹤双拼的用户 |

| 自然码双拼 | 双拼方案，适合自然码输入习惯用户 |

---

## ✨ 亮点功能

### 1️⃣ Emoji 增强
- 在打字的同时，显示 Emoji 表情候选  
- 仅支持部分精选 Emoji，作为辅助亮点功能  

### 2️⃣ 前后缀和邮箱等便携输入
- 通过快捷方式快速输入常用符号  
- 增强高频使用的精选特殊符号输入体验
- 支持邮箱域名速打
- 支持软件、媒体、网址等常见前后缀
  
### 3️⃣ 词条过滤  
- 更纯净的词库，尽可能避免一些不合时宜的词语
